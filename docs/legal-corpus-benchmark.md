# 法规库对标清单（vs 北大法宝 / 威科先行 劳动法目录）

> **用途**：把现有法规库对标业内权威库（北大法宝 pkulaw.com / 威科先行 law.wkinfo.com.cn）的
> 劳动法目录，列出覆盖缺口，作为入库排期与人工核验的依据。
> **来源边界**：法规**条文本身从官方免费源抓取**（flk.npc.gov.cn / gov.cn / 人社部 / 最高法官网，
> 见 `ingest_law.py`），**不从北大法宝/威科先行下载**（付费登录墙，ToS + 著作权汇编保护）。
> 两库**只作人工对标基准**：标 ⚠ 的条目，现行有效性 / 准确文号 / 是否被修订替代，须由运营/律师
> 对照北大法宝或威科先行确认后再定 fetch_url 入库。
> **状态**：✅ 已入库 ｜ ⬜ 待抓取（官方源）｜ ⚠ 待对标确认（文号/时效）

## 三、入库次序（建议）
1. 全国性 ⬜（官方源齐、争议高频）：工会法 → 保障农民工工资支付条例 → 劳动保障监察条例 →
   劳务派遣暂行规定 → 失业保险条例 → 工伤认定办法 → 禁止使用童工规定 → 司法解释（二）
2. ⚠ 条目：先由运营/律师对照北大法宝/威科先行确认文号与时效，再补 manifest 入库
3. 地方性：按目标辖区逐个 websearch --official → ingest_law

每部入库流程：`websearch --official` 找官方源 → 写 manifest（fetch_url）→ `ingest_law.py <slug>` →
条号自检 → `verify_articles.py --check` → 人工对照北大法宝 → `--approve --benchmark`（G1）。
